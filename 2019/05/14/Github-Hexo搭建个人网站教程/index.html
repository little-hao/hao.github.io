Github+Hexo搭建个人网站教程

通过带有预览样式的Markdown编辑器实时预览书写的博文样式，
也可以通过命令 hexo s –debug 在本地浏览器的localhost:4000 预览博文效果。
写好博文并且样式无误后，通过hexo g、hexo d 生成、部署网页。
随后可以在浏览器中输入域名浏览
